🌿 GreenMind — AI Agriculture Assistant
🔹 1. Introduction

GreenMind is a web-based AI assistant for farmers that helps them make better agricultural decisions using AI, images, and user inputs.

It allows users to:

Detect plant diseases
Get crop recommendations
Receive fertilizer and medicine advice
Choose crops based on location and season
Manage irrigation and soil pH
🔹 2. Problem Statement

Farmers often face problems like:

Not identifying plant diseases correctly
Choosing wrong crops for climate/soil
Improper fertilizer or irrigation usage

GreenMind solves this by providing instant AI-based guidance.

🔹 3. Key Features
📌 Multi-Function System (6 Modes)

The system provides six main features:

Disease Detection
→ Identify plant diseases using image + text
Crop Recommendation
→ Suggest best crops based on conditions
Medication Advice
→ Suggest treatments and chemicals
Fertilizer Guide
→ Recommend NPK ratio and usage
Location & Season (Weather-based)
→ Suggest crops based on climate
Irrigation & pH Advisor
→ Manage water and soil acidity
🔹 4. User Interaction Flow
User selects a mode
Enters details (text input)
Optionally uploads image or uses live camera
Clicks Analyze
AI processes and gives results
🔹 5. Technologies Used
Frontend: HTML, CSS, JavaScript
AI Model: Google Gemini API
Computer Vision: Camera + Image Upload
UI Features:
Live camera scanning
Image preview
Interactive sliders (pH, irrigation)
🔹 6. Special Features
📷 Image + Camera Support
Upload plant images
Live camera scanning with capture
🌡️ Smart Input Controls
Soil pH slider (0–14 scale)
Irrigation frequency selection
Water amount control
⚡ Real-Time AI Response
Sends user data to AI
Displays result instantly
🔹 7. How AI Works (Backend Logic)
User input is converted into a structured prompt
Example:
Disease → identify + cause + treatment
Crop → suggest best crops
If image is present → sent along with text
AI generates detailed agricultural advice
🔹 8. Advantages
Easy to use for farmers
Works with both text and images
Covers multiple agricultural needs in one platform
Provides fast and practical solutions
🔹 9. Limitations
Depends on internet connection
AI accuracy may vary
Not a replacement for expert agronomists
🔹 10. Conclusion

GreenMind is a smart agriculture assistant that combines:

AI
Computer vision
User-friendly interface

to help farmers make better, faster, and more informed decisions.
